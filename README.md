# Frieren

一款webshell管理器。

使用方式：使用IDEA打开viewmain类运行。

# V0.0.1

支持生成三种webshell

- php（一句话木马）
- jsp内存马（仅可执行单个包含空格的命令，等找到更好的马会改进）
- spring内存马（templatesImpl加载字节码）

php支持一键上黑页（默认目录）
